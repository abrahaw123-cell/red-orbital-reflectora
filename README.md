# Red Orbital Reflectora 

Sistema de energía solar orbital que captura y refleja luz solar 
desde órbita baja (LEO 400-2000 km) hacia receptores terrestres, 
sin generar electricidad en órbita.

## Sistema Ei — 20 Ecuaciones en 4 Capas

| Grupo | Ecuaciones | Peso |
|-------|-----------|------|
| 🪐 Gravedad | HCW, Kepler, Newton, Euler-Lagrange, Roche | α = 0.45 |
| ☀️ Luz | Reflexión, Lambert, Fresnel, Difracción, Rad. | β = 0.10 |
| 🧪 Materiales | Hooke, Maxwell, Ohm, Calor, Young | γ = 0.30 |
| 🤖 IA | Kalman, Consenso, Costo, Fourier | δ = 0.15 |

El eigenvalor **Ei = αEgrav + βEluz + γEmat + δEIA** unifica las 4 capas 
como indicador de estado de cada nodo satelite.
